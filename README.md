# springboot-dubbox

基于spring-boot dubbox搭建的分布式系统示例

- 前后端分离
- 前端管理项目 https://github.com/lufengc/springboot-dubbox-web
- spring-security基于角色的权限管理
- JWT (Json Web Token) with Spring Security
- 统一异常管理
- spring-data-redis整合
- 整合swagger-ui接口文档
- 多profile管理
- maven项目管理
- Mybatis多数据源
- logback记录日志
- 客户端用户注册、登录、修改信息、上传头像、忘记密码接口
- 接口访问次数限制
- okhttp3第三方接口调用
- druid sql监控
